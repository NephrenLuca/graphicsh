### **Slide 10**
**标题**：Project 1.1 曲线的绘制 – 二维曲线  

如果只绘制曲线，计算点坐标即可。但动画和曲面建模需要更多信息（如切向量、法向量）。

考虑汽车沿平面曲线 q(t) 运动：位置 V = q(t)，单位切向量 T = q'(t)/||q'(t)||。  
若定义法向量 N = T'/||T'||（即二阶导），存在问题：拐点处法线方向翻转，直线处除零。  

因此采用不同方法：引入次法线 B，在二维中固定指向正 z 方向，则 N = B × T。

### **Slide 12**
**标题**：Project 1.1 曲线的绘制 – 三维曲线  

三维中不能固定 B（曲线可能旋转导致 T = B）。  
采用递归更新方法，沿离散点 t1, t2, ... 计算：

在第 i 步：
- 位置 V_i
- 切向量 T_i = q'(t_i)/||q'(t_i)||
- 临时法线 N_tmp = N_{i-1} - (N_{i-1}·T_i) T_i
- 然后 N_i = normalized(N_tmp)
- B_i = normalized(T_i × N_i)

初始化：选择任意 B_0（不平行于 T_1），如 B_0 = normalized((0,0,1) × T_1)，然后计算 N_1 = normalized(B_0 × T_1)。

### **Slide 13**
**标题**：Project 1.1 曲线的绘制 – 三维曲线（坐标几何意义）  

三个向量定义沿曲线的局部坐标系。  
变换矩阵为：
```
[ N_x  B_x  T_x  V_x ]
[ N_y  B_y  T_y  V_y ]
[ N_z  B_z  T_z  V_z ]
[ 0    0    0    1   ]
```
其中 N、B、T 组成旋转矩阵，V 为平移。

### **Slide 20**
**标题**：Project 1.2 曲面的绘制 – 旋转曲面  

定义：绕正 Y 轴逆时针旋转 XY 平面上的曲线。  
通过均匀旋转采样复制曲线点生成顶点。还需定义法线和面。  

法线变换：顶点用变换矩阵 M 变换时，法线需用 M 的左上 3×3 子矩阵的逆转置变换。  
方向问题：曲线法线（红色）应指向遍历方向的左边（N = B × T）。  
对于旋转曲面，需反转曲线法线方向以获得向外曲面法线。  
假设轮廓曲线总是在 Y 轴左侧（所有点 x ≤ 0），以避免法线方向歧义。

### **Slide 25**
**标题**：Project 1 拓展 – 曲面的闭合问题  

问题：闭合曲线首尾坐标系不对齐（起点和终点位置、切线相同但法线不同）。  
解决方法：检测误差，然后插值扫描曲线开始和结束之间的旋转差值，并将该旋转逐步应用到沿曲线的坐标系上。  
完成后可无缝显示 `weirder.swp`，且不影响非闭合曲线。

### **Slide 26**
**标题**：Project 1 拓展 – 曲面的闭合问题（插值解决）  

将扫掠曲线的法向量 N 和次法线 B 绕切向量 T 旋转 θ = α / surf_size 度，其中 α 为起始法线与结束法线的夹角。  
使用罗德里格旋转公式。简化后：N' = cosθ N + sinθ B。
